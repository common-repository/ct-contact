== Description ==

Want to display your personal or business contact information? Then this awesome lil' contact widget plugin is for you. It includes Title, Company, Street, City, State, Postal, Country, Email & a View More Link.

== Installation ==

1. Upload the /ct-contact/ folder to the /wp-content/plugins/ directory.
2. Activate the CT Contact plugin through the "Plugins" menu in WordPress.
3. Configure your settings via Appearance > Widgets > CT Contact.
4. And you're good to go!
